RECOMMENDED IMAGE SPECIFICATIONS:
---------------------------------
- Format: JPG or PNG
- Size: 500x500px minimum (square aspect ratio)
- Resolution: 72-150 DPI
- File size: Under 500KB for optimal loading
- Background: Any (site will handle styling)

TIPS:
-----
1. Use high-quality, clear photos
2. Ensure faces are well-lit and visible
3. Professional or gaming-themed photos work best
4. Consider using consistent style across all photos
5. PNG format recommended if you want transparent backgrounds

If photos are not available, placeholder graphics will be shown automatically.
